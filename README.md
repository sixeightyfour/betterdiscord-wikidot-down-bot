# betterdiscord-wikidot-down-bot
A plugin for BetterDiscord.

## Features:
* **No-Click Status Updates:** The color of the exclamation mark icon corresponds to the last request's results (green for up, red for down, yellow for slow).
* **Down Notifications:** Provides a notification for the user when the site goes down, as well as when it comes back up.
* **History:** In-menu list of previous results, with timestamps.
* **Auto-Check:** Togglable automatic status checks at intervals of the user's choice.
* **And Much More!**

## Setup Instructions:
1. Download [BetterDiscord](https://betterdiscord.app/). Follow the instructions to install it (takes 1-3 minutes).
2. [Download the Plugin](https://github.com/sixeightyfour/betterdiscord-wikidot-down-bot/blob/main/SCPWikiDownDetector.plugin.js) from this repo (click on the downward arrow on the top right of the code block).
3. Go to your Discord settings, and scroll to the bottom of the menu. There should now be a section called "BetterDiscord". Select "Plugins".
4. In the Plugins menu, press the folder icon in the top left. This should open up a folder on your computer.
5. Put the downloaded plugin file into that folder. Then, click back into Discord, and press Ctrl+R (refreshes the app).
6. Go back to the plugin menu. You should now see SCPWikiDownDetector in the center of your screen; make sure the slider is set to Enabled.
7. Click the double exclamation mark icon to fine tune your settings. Have fun!
